Block Level Elements

Block-level elements occupy the full width available, creating a new line before and after the element. They are typically used for larger structures or sections of a webpage

  • Div Tag
  • Heading Tags (h1-h6)
  • Paragraph Tag
  • Unordered List (ul)
  • Ordered List (ol)
  • List Items (li)
  • Header Tag
  • Footer Tag
  • Section Tag
  • Article Tag
  • Form Tag
  • Table Tag

Inline Level Elements

Inline-level elements do not start on a new line and only take up as much width as necessary. They are typically used for smaller pieces of content within block-level elements.

  • Span Tag
  • Anchor Tag
  • Image
  • Strong Tag
  • Emphasized Tag
  • Break Tag (br)
  • Label Tag
